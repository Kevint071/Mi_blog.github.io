Definicion Características Descripcion gráfica
Data science Se encargan de la toma de decisiones basadas en datos e incorporar datos a los productos de software. Obtener, limpiar y procesar datos, además de utilizar modelos de machine learning para generar predicciones sobre los datos.
Data Analyst Se enfoca en analizar datos en las bases de datos de la organización, analizándolos y reportándolos en gráficas y tablas. Analizar los datos para identificar patrones y tendencias que se puedan convertir en información accionable.
Data Engineer Crean flujos de extracción, transformación y carga de datos para que analistas y científicos de datos puedan recuperar fácilmente los datos. Extraer datos de diferentes fuentes como bases de datos estructuradas y no estructuradas, API y archivos.
